## 项目信息

- **项目名称**: PP-Alpha-Agent
- **团队**: PredP.red AI Operations Team
- **赛道**: Agent Trade Kit

## 核心创新点

### 1. 预测驱动的交易信号

传统的交易策略依赖技术指标和基本面分析。PP-Alpha-Agent 创新性地将 **预测市场** 作为独特的信号来源：

- Polymarket 上的预测价格代表群体智慧对未来事件的概率判断
- 当大量真金白银押注某个结果时，价格变动领先于市场反应
- 我们将这种"群体预测"转化为可执行的交易信号

### 2. AI 深度分析

利用 Claude AI 分析预测市场信号：

- 理解事件背景和市场影响
- 评估风险和收益预期
- 生成结构化的交易决策

### 3. 全自动化执行

通过 OKX Agent Trade Kit 实现闭环交易：

- 自动执行买卖
- 智能止盈止损
- 仓位风险管理

### 技术栈

- **数据源**: Polymarket Gamma API + CLOB API
- **AI 引擎**: Claude (Anthropic)
- **交易执行**: OKX Agent Trade Kit (MCP Protocol)
- **语言**: TypeScript
- **运行时**: Node.js 18+
